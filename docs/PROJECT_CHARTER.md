# Project Charter

O MCL Piloto Classe II v0.1.0 materializa, em software demonstrativo, a tese de continuidade informacional entre sistemas e registros logísticos.

## Objetivo

Permitir que uma necessidade sintética de 200 pares de coturnos seja acompanhada do registro inicial até a entrega, preservando fonte, horário, confiança, vínculos e auditoria.

## Fora de escopo

- integração real com sistemas oficiais;
- dados pessoais reais;
- contratos, saldos, rotas, fornecedores ou capacidades reais;
- decisão administrativa autônoma.

## Critério de sucesso

Uma pessoa deve ler o QR de uma caixa de coturnos e reconstruir necessidade, cobertura, crédito, aquisição, lote, remessa, entrega, divergência, correção, origem dos dados e trilha de auditoria.
