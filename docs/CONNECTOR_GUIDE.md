# Guia de Conectores

Conectores da v0.1.0 são simulados:

- `SIM-NECESSIDADES`: saudável.
- `SIM-FINANCEIRO`: atrasado.
- `SIM-TRANSPORTE`: falha.

Nenhum conector acessa sistema real. Integrações futuras exigem autorização, contrato de dados, credenciais seguras, trilha de auditoria e classificação.
