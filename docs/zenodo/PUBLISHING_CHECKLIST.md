# Checklist de Publicação Zenodo

- [ ] revisar dados sintéticos;
- [ ] revisar ausência de segredos;
- [ ] confirmar licença;
- [ ] confirmar autoria;
- [ ] confirmar relação com trabalho acadêmico;
- [ ] conferir versão;
- [ ] conferir checksum;
- [ ] criar release Git;
- [ ] criar release GitHub, quando disponível;
- [ ] carregar ZIP;
- [ ] carregar documentação principal;
- [ ] inserir related identifier;
- [ ] reservar ou emitir DOI;
- [ ] atualizar README com DOI somente depois de emitido.
