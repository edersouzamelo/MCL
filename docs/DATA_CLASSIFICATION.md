# Classificação de Dados

Permitido na v0.1.0:

- dados sintéticos;
- organizações fictícias;
- usuários demonstrativos;
- fornecedores e transportes sintéticos;
- documentos referenciados sinteticamente.

Proibido:

- dados pessoais reais;
- saldos, contratos, rotas, estoques, capacidades ou fornecedores reais;
- credenciais;
- informação classificada;
- vulnerabilidades ou interfaces internas.
