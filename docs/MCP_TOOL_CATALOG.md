# Catálogo MCP Futuro

Ferramentas previstas, somente leitura:

- `consultarSituacaoMaterial()`
- `reconstruirTrajetoriaLote()`
- `compararNecessidadesEstoque()`
- `consultarCreditoEmpenho()`
- `localizarInstrumentoAquisicao()`
- `executarAnaliseMulticriterio()`
- `listarDivergencias()`
