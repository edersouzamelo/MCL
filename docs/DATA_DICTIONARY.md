# Dicionário de Dados

Campos comuns:

- `sourceSystem`: origem declarada do registro.
- `sourceRecordId`: identificador na origem.
- `occurredAt`: quando o fato ocorreu.
- `recordedAt`: quando o MCL registrou.
- `confidence`: grau de confiança de 0 a 1.
- `schemaVersion`: versão do esquema.
- `dataNature`: `validado`, `informado`, `calculado`, `estimado`, `divergente` ou `pendente`.

Entidades principais: `Need`, `Credit`, `AcquisitionInstrument`, `Commitment`, `NeedCoverage`, `Lot`, `LogisticsUnit`, `Shipment`, `LogisticsEvent`, `Divergence`, `AuditLog`.
