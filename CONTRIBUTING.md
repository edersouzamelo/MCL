# Contribuição

Este protótipo é demonstrativo e científico. Antes de contribuir:

- não inclua dados reais, classificados, credenciais ou informações operacionais;
- mantenha todo dado demonstrativo marcado como sintético;
- preserve eventos e auditoria como append-only;
- não apresente simuladores como integrações reais;
- rode `pnpm lint`, `pnpm typecheck`, `pnpm test` e `pnpm build`.

Mudanças de escopo institucional, licença, DOI, afiliação ou endosso exigem decisão expressa do autor.
